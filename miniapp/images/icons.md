# 小程序图标说明

微信小程序需要实际的图片文件作为 tabBar 图标。请将以下图标文件放置在此目录：

## 需要的图标文件

| 文件名称 | 尺寸 | 用途 |
|----------|------|------|
| home.png | 81x81 | 首页默认图标 |
| home-active.png | 81x81 | 首页选中图标 |
| recycle.png | 81x81 | 回收默认图标 |
| recycle-active.png | 81x81 | 回收选中图标 |
| order.png | 81x81 | 订单默认图标 |
| order-active.png | 81x81 | 订单选中图标 |
| points.png | 81x81 | 积分默认图标 |
| points-active.png | 81x81 | 积分选中图标 |
| profile.png | 81x81 | 我的默认图标 |
| profile-active.png | 81x81 | 我的选中图标 |

## 图标规范

- 格式：PNG
- 尺寸：81px × 81px
- 背景：透明
- 未选中状态：灰色 (#8E8E93)
- 选中状态：主题色 (#2563EB)

## 生成方式

您可以使用以下方式生成图标：

1. 使用图标库（如 iconfont）下载 PNG 图标
2. 使用设计工具（如 Figma、Sketch）制作
3. 使用在线工具生成

## 临时解决方案

如果暂时没有图标文件，可以：

1. 使用纯色占位图
2. 使用 base64 编码的小图标
3. 注释掉 app.json 中的 tabBar 配置，使用自定义 tabBar

## 注意事项

- 图标文件必须存在，否则小程序无法正常运行
- 图标尺寸过大可能影响加载性能
- 建议使用压缩后的 PNG 格式
