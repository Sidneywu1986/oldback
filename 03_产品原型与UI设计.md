## 1. 全局UI规范

### 1.1 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（Primary） | `#1677FF` | 按钮、链接、选中态、图标高亮 |
| 主色悬停 | `#4096FF` | 按钮悬停状态 |
| 主色激活 | `#0958D9` | 按钮点击状态 |
| 背景色 | `#F5F7FA` | 页面全局背景 |
| 卡片背景 | `#FFFFFF` | 内容卡片、弹窗背景 |
| 成功色 | `#52C41A` | 审核通过、已入库状态 |
| 警告色 | `#FAAD14` | 待审核状态 |
| 危险色 | `#FF4D4F` | 驳回、删除、异常状态 |
| 文本主色 | `#262626` | 标题、正文 |
| 文本次要 | `#595959` | 辅助说明文字 |
| 文本占位 | `#BFBFBF` | 输入框占位符 |
| 边框色 | `#D9D9D9` | 表格边框、分割线 |
| 禁用背景 | `#F5F5F5` | 禁用状态背景 |
| 禁用文字 | `#BFBFBF` | 禁用状态文字 |

### 1.2 尺寸与间距

| 项目 | 值 | 换算说明 |
|------|------|----------|
| 卡片圆角 | `12px` | 统一圆角，视觉柔和 |
| 按钮高度 | `48px`（主按钮） | 对应移动端 `88rpx` |
| 按钮高度（小） | `32px` | 表格内操作按钮 |
| 输入框高度 | `40px` | 表单统一高度 |
| 页面内边距 | `24px` | 卡片与页面边缘间距 |
| 卡片内边距 | `20px` | 卡片内容区域间距 |
| 模块间距 | `16px` | 卡片与卡片之间间距 |
| 表格行高 | `compact: 48px / comfortable: 64px` | 两档可切换 |

### 1.3 字体规范

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 页面标题 | `24px` | 600 | `32px` | 页面顶部大标题 |
| 卡片标题 | `16px` | 600 | `24px` | 卡片内模块标题 |
| 正文 | `14px` | 400 | `22px` | 表格内容、表单文字 |
| 辅助文字 | `12px` | 400 | `20px` | 时间戳、提示信息 |
| 统计数字 | `32px` | 700 | `40px` | 看板大数字展示 |

> 字体栈：`system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
> 统一使用系统默认字体，不引入外部字体文件，确保跨平台兼容性。

### 1.4 图标规范

- **风格**：线性简约风格，2px描边
- **尺寸**：表格内操作图标 `16px`，按钮内图标 `14px`，导航图标 `20px`
- **颜色**：跟随容器文字颜色，激活态使用主色 `#1677FF`
- **统一使用** `iconfont` 或 `Ant Design` 图标库，禁止混用多种风格

### 1.5 图片规范

- **形状**：矩形圆角 `8px`，无边框
- **缩略图尺寸**：旧件照片预览 `120px × 120px`
- **大图预览**：支持点击放大，最大宽度 `800px`
- **占位图**：加载中显示灰色背景 `#F0F0F0` + 图标

#### 2.1.2 字段说明

| 字段 | 类型 | 必填 | 规则 | 占位提示 |
|------|------|------|------|----------|
| 账号 | 文本输入 | 是 | 长度 4-20 位，支持字母/数字/下划线 | 请输入账号 |

#### 2.1.3 交互规则

1. **聚焦态**：输入框边框变为 `#1677FF`，显示蓝色阴影 `0 0 0 2px rgba(22,119,255,0.2)`
2. **失焦校验**：离开输入框时即时校验，错误信息在输入框下方红色文字显示 `#FF4D4F`
3. **登录按钮**：
   - 未输入完整时：禁用态，背景 `#BFBFBF`
   - 可点击时：主色背景 `#1677FF`，悬停 `#4096FF`
   - 点击后：加载中状态，按钮文字变为「登录中...」+ 加载图标
4. **登录成功**：跳转至 `/dashboard`，并在 URL 中记录 `redirect` 参数以便返回原页面
5. **登录失败**：错误提示 Toast 显示，3 秒后自动消失
6. **记住我**：勾选后 Token 有效期 7 天，不勾选则 Session 关闭即失效
7. **键盘快捷键**：按 `Enter` 键触发登录按钮

   - 点击「订单号」跳转 `/order/detail?id=xxx`

   - 日期范围快捷选项：今日 / 近7天 / 近30天 / 本月 / 上月

   - 「紧凑」模式行高 `48px`，适合数据量大场景
   - 「舒适」模式行高 `64px`，适合需要查看详情场景
   - 切换即时生效，无刷新动画
3. **操作按钮**：
   - 「查看」：跳转 `/recycle/detail?id=xxx`
   - 「审核」：仅待审核状态显示，点击弹出审核确认弹窗
   - 「驳回」：仅待审核状态显示，点击弹出驳回原因输入弹窗

   - 勾选表头复选框全选当前页
   - 选中后顶部显示批量操作栏：「批量审核通过」「批量驳回」
5. **分页规则**：
   - 默认每页 `20` 条，可切换 `10/20/50/100`
   - 页码按钮最多显示 `7` 个（含省略号）
   - 直接输入页码跳转，校验范围 `1-总页数`

### 3.1 弹窗规范

| 类型 | 宽度 | 用途 | 按钮组合 |

**弹窗行为规则**：

1. **遮罩层**：黑色半透明 `rgba(0,0,0,0.45)`，点击遮罩层可关闭弹窗（确认弹窗除外）
2. **打开动画**：从中心缩放淡入 `transform: scale(0.9) → scale(1)`，持续时间 `200ms`，缓动 `ease-out`
3. **关闭动画**：反向缩放淡出，持续时间 `150ms`
4. **确认弹窗不可通过点击遮罩关闭**，必须点击按钮
5. **弹窗层级**：统一使用 `z-index: 1000`，叠加弹窗递增
6. **滚动锁定**：弹窗打开时背景页面禁止滚动 `overflow: hidden`

### 3.2 表单校验规则

**校验触发时机**：

| 场景 | 触发时机 | 提示方式 |
|------|----------|----------|
| 即时校验 | 输入框失焦（blur） | 输入框下方红色文字提示 |
| 提交校验 | 点击提交/保存按钮 | 汇总所有错误，第一个错误项自动聚焦 |
| 异步校验 | 输入后延迟 300ms | 如用户名重复、编码唯一性 |

**校验提示样式**：

```
┌─────────────────────────────────────┐
│ [输入框内容]                        │
│ └─────────────────────────────────┘

└─────────────────────────────────────┘
```

**通用校验规则**：

| 规则 | 正则/逻辑 | 错误提示文案 |
|------|-----------|--------------|
| 必填 | 值不为空 | `请输入{字段名}` / `请选择{字段名}` |
| 最小长度 | `value.length >= min` | `{字段名}最少{min}个字符` |
| 最大长度 | `value.length <= max` | `{字段名}最多{max}个字符` |
| 正整数 | `/^[1-9]\d*$/` | `{字段名}必须为正整数` |
| 手机号 | `/^1[3-9]\d{9}$/` | `请输入正确的手机号码` |
| 日期范围 | 结束日期 ≥ 开始日期 | `结束日期不能早于开始日期` |
| 数字范围 | `min <= value <= max` | `{字段名}需在{min}~{max}之间` |

3. **加载状态**：操作进行中按钮显示加载动画，禁止重复点击

### 3.4 分页规则

**分页组件规范**：

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│  共 {total} 条记录    每页 [20▼] 条                         │
│                                                            │
│  [首页]  [上一页]  [1]  [2]  [3]  ...  [10]  [下一页]  [尾页]│
│                                                            │
│  跳转到 [    ] 页    [确定]                                 │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

**分页参数**：

| 参数 | 默认值 | 可选值 | 说明 |
|------|--------|--------|------|
| pageSize | `20` | `10, 20, 50, 100` | 每页记录数 |
| current | `1` | `1 ~ 总页数` | 当前页码 |
| total | — | — | 总记录数（由接口返回） |

**分页交互规则**：

1. **切换 pageSize**：当前页重置为第 `1` 页，触发重新查询
2. **页码按钮**：最多显示 `7` 个页码按钮，当前页居中，前后各显示省略号
3. **快速跳转**：输入页码后按 `Enter` 或点击「确定」跳转
4. **边界控制**：输入页码 `< 1` 时跳转第 `1` 页，`> 总页数` 跳转最后一页
5. **禁用状态**：第 `1` 页时「首页」「上一页」禁用；最后一页时「尾页」「下一页」禁用
6. **URL 同步**：页码和 pageSize 同步到 URL 参数 `?page=1&size=20`，支持浏览器前进后退
